# 🛋️ House Staff Furniture Store

Modern furniture store web app built with React and Vite.

## ✨ Overview

House Staff is a responsive e-commerce demo for browsing furniture, filtering products by category, viewing product details, and managing a shopping cart. The project is suitable as a portfolio piece for clients, recruiters, or employers looking for practical frontend skills.

## 🚀 Key Features

- 🪑 Product catalog with furniture cards
- 🗂️ Category filtering for chairs, tables, sofas, lighting, and armchairs
- 🛒 Shopping cart with add, remove, and total price calculation
- 🔍 Product detail modal with image preview
- 📄 Simple navigation pages: About, Contacts, and Account
- ⚡ Fast development workflow powered by Vite

## 🧰 Tech Stack

- ⚛️ React 19
- ⚡ Vite 7
- 🎨 CSS
- 🧩 React Icons

## 📦 Getting Started

```bash
cd shop
npm install
npm run dev
```

## 🏗️ Build

```bash
cd shop
npm run build
```

## 👤 For Clients & Employers

This project demonstrates component-based UI development, state management for catalog and cart interactions, clean project structure, and the ability to build a polished user-facing storefront experience.
